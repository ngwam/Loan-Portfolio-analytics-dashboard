# Bank Loan Performance Dashboard

Overview

This project is an interactive Power BI dashboard designed to analyze bank loan performance and provide insights into lending activity, borrower behavior, loan quality, and repayment trends.

The dashboard enables stakeholders to monitor key lending metrics, evaluate portfolio health, identify high-performing loan segments, and support data-driven decision-making.

# Business Problem

Financial institutions need visibility into loan performance to effectively manage risk, monitor repayments, and identify lending opportunities.

This dashboard was developed to answer critical business questions related to loan applications, funded amounts, repayments, borrower characteristics, and loan status.

# Business Questions

The dashboard answers the following questions:

- How many loan applications have been submitted?

- What is the total funded loan amount?

- How much money has been received from borrowers?

- What percentage of loans are classified as good or bad loans?

- Which states have the highest lending activity?

- What are the average interest rates and debt-to-income ratios?

- Which borrower segments demonstrate stronger loan performance?
